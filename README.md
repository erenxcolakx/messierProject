## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation
Clone the repository:
   ```bash
   git clone https://github.com/erenxcolakx/messierProject.git
   ```
Install dependencies:
   ```bash
   cd messierProject
   npm install
   ```
Start the application:
   ```bash
   npm run dev
   ```
